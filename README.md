# Scientific Calculator

A command-line scientific calculator built with Python.

---

## Features

### Basic Arithmetic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### Scientific Functions
- Square Root
- Power
- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Cotangent (cot)

### Extra Features
- Calculation history tracking
- Continuous execution until the user chooses to exit
- Protection against division by zero
- Input validation using error handling
- Degree-based trigonometric calculations

---

## Technologies Used

- Python 3
- Built-in Math Module

---

## How to Run

1. Install Python 3
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run:

```bash
python scientific_calculator.py
